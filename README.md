<ul>
<li>Contains water and lava tanks</li>
<li>2x2 crafting grid in the center</li>
<li>Used to craft various stones</li>
<li>Must pray while holding a stone to enable its special ability (fills durability)</li>
<li>Will need to refill durability on stones / armor / weapons at an alter periodically</li>
</ul>
